========================================
  OBSIDIAN — INSTRUÇÕES DE USO
========================================

REQUISITO
---------
- Python 32-bit instalado (python.org)
  Marcar "Add Python to PATH" na instalação


PASSO A PASSO — TODO DIA
-------------------------

1. Abra a pasta do sistema

2. Clique duas vezes em:  INICIAR.BAT
   → Fica rodando em segundo plano
   → Envia os dados a cada 15 minutos
   → NÃO feche essa janela

3. Clique duas vezes em:  DASHBOARD.BAT
   → Abre o browser com o painel
   → Se fechar o browser, dê dois cliques novamente


ABAS DO DASHBOARD
-----------------
  Sales Ledger  → Calendário de vendas por dia
  Produtos      → Gráfico e tabela de vendas por produto
  Estoque       → Projeção de compras


AO DESLIGAR O COMPUTADOR
-------------------------
Pode fechar as janelas normalmente.
Na próxima vez, repita o Passo 2 e 3.


PROBLEMAS COMUNS
----------------
"Erro ao conectar no Firebird"
  → Verifique se o MegaFlex está aberto
  → Verifique se o computador está na rede

Dashboard não carrega dados
  → Verifique conexão com internet
  → Recarregue a página (F5)
